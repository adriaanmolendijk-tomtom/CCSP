# Domain 13: Security as a Service

## Overview

SecaaS benefits
- Cloud computing
- Staffing and expertise
- Intelligence sharing
- Deployment flexibility
- Insulation
- Scaling and cost

Concerns
- Lack of visibility
- Regulation differences
- Handling of regulated data
- Data leakage
- Changing providers
- Migration to SecaaS

Categories
- IAM
- CASB
- Web security
- Email security
- IDS/IPS
- Encryption and key management
- BC/DR
- DDoS protection
